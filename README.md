# GRC Risk Toolkit

This repository documents a collection of small tools designed to simulate core Governance, Risk and Compliance (GRC) practices.

The toolkit demonstrates simplified implementations of common risk management activities such as compliance assessment, control testing, and risk scoring.

## Tools Included

### 1. Regulatory Compliance Checklist
Evaluates compliance practices and identifies potential control gaps.

### 2. Internal Control Testing Tool
Simulates internal control testing using sample-based evaluation.

### 3. Risk Assessment Calculator
Calculates risk score using a standard Impact × Likelihood methodology.

### 4. Risk Register Analyzer
Analyzes a risk register and identifies high priority risks.

## Risk Methodology

The toolkit uses a simplified risk matrix:

Risk Score = Impact × Likelihood

| Score | Risk Level |
|------|-------------|
|1–4|Low|
|5–9|Medium|
|10–16|High|
|17–25|Critical|

## Objective

The goal of this project is to demonstrate practical applications of risk management concepts using simple Python tools.

This portfolio is intended for learning and demonstration purposes in Governance, Risk and Compliance environments.
